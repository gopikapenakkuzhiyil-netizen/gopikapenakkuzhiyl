# gopikapenakkuzhiyl# 🚀 Task Manager

A simple Task Manager application built with Python that allows users to create, update, delete, and view daily tasks.

## 📖 Overview

This project demonstrates basic CRUD (Create, Read, Update, Delete) operations using Python. It is designed for beginners who want to learn project structure and GitHub documentation.

## ✨ Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Simple command-line interface

## 🛠️ Technologies Used

- Python 3.11
- SQLite
- Git & GitHub

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/task-manager.git
```

Navigate to the project directory:

```bash
cd task-manager
```

Install the required packages:

```bash
pip install -r requirements.txt
```

## ▶️ Running the Project

Start the application:

```bash
python main.py
```
